# 麻将计分工具

一个专为麻将比赛设计的纯前端计分记录工具，支持多平台使用，数据本地存储。

## 功能特点

### 🎯 核心功能
- **选手管理**：支持2-4名选手参与，自动生成彩色头像
- **实时计分**：单局计分，自动计算总分，确保总和为0
- **历史记录**：完整的游戏记录，支持查看和删除
- **数据持久化**：使用本地存储，数据不会丢失

### 📱 设备适配
- **手机浏览器**：针对移动端优化，支持Android和iOS
- **平板设备**：适配各种尺寸的平板电脑
- **桌面电脑**：支持桌面浏览器访问

### 🎮 界面功能

#### 主界面（初始状态）
- 添加/删除选手（最多4人）
- 选手头像自动生成
- 开始计分按钮（至少2人才能开始）

#### 计分中界面
- 显示各选手当前总分
- 历史局数详细记录
- 计分和结算按钮

#### 计分界面
- 选择胜负状态
- 输入得分
- 自动计算最后一名选手得分
- 确保总分为0的验证

#### 历史记录界面
- 查看进行中和已结束的游戏
- 按月份筛选记录
- 删除历史记录
- 点击进入游戏详情

## 技术栈

- **Vue 3**：主框架，使用Composition API
- **Vue Router**：路由管理
- **Element Plus**：UI组件库
- **Vite**：构建工具
- **LocalStorage**：数据存储

## 使用说明

### 开始新游戏
1. 在主界面点击"+"按钮添加选手
2. 输入选手姓名并确认
3. 添加至少2名选手
4. 点击"开始计分"按钮

### 记录单局得分
1. 在计分中界面点击"计分"按钮
2. 为每位选手选择胜负状态
3. 输入得分（最后一名选手自动计算）
4. 确认提交（总分必须为0）

### 结算游戏
1. 在计分中界面点击"结算"按钮
2. 确认结算后游戏结束
3. 记录将保存到历史记录中

### 查看历史记录
1. 通过菜单进入历史记录页面
2. 切换"进行中"和"已结束"标签
3. 选择月份筛选记录
4. 点击记录查看详情或删除

### 数据结构
```javascript
// 游戏对象结构
{
  id: String,           // 游戏唯一ID
  players: Array,       // 选手列表
  rounds: Array,        // 局数记录
  status: String,       // 'playing' | 'finished'
  startTime: Number,    // 开始时间戳
  endTime: Number       // 结束时间戳（可选）
}

// 选手对象结构
{
  id: String,           // 选手唯一ID
  name: String,         // 选手姓名
  color: String,        // 头像颜色
  totalScore: Number    // 当前总分
}

// 局数记录结构
{
  roundNumber: Number,  // 局数
  timestamp: Number,    // 时间戳
  scores: Object        // 选手得分 {playerId: score}
}
```
